# arma-lisp

A lisp dialect that compiles to sqf.

## Expressions
- [x] if 
- [x] def
- [x] basic math expressions [+, -, *, /, %]
- [x] do 
- [x] logical operators [and, or, =, <, <=, >, >=, !=]
- [x] arrays
- [ ] for
- [ ] while
- [x] fn
- [ ] foreach
- [ ] case
- [ ] exitwith
- [ ] waituntil
- [ ] default
- [x] function call syntax
